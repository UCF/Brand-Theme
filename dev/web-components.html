Widgets

What's a Widget?

Widgets are code snippets that display information that is pulled from other sources than the website on which the widget is used. Web widgets work like a mini-application that you use to provide information to visitors on websites.

UCF Map Widget

The UCF Map widget pulls data from https://map.ucf.edu and allows you to add a mini map to your website. The embedded map is customizable and can include multiple locations, be adjusted for width/height and display either the Google Map or UCF Illustrated map version. For more information on using the UCF Map widget, visit https://map.ucf.edu/widget/. Example Single Location Map

UCF Today Widget

The UCF Today widget can be used to add a news feed from UCF Today on your website. For more information and configuration details, visit http://today.ucf.edu/widget/. Example Unstyled Source

UCF Events Widget

The events widget pulls content from the UCF Event System. Step-by-step instructions on how to add and configure this widget can be found at http://events.ucf.edu/help/#events-widget.

CSS Frameworks

For designing and building websites, UCF Marketing has used two cascading stylesheet (CSS) and javascript frameworks.We originally built sites on Blueprint CSS which offered a CSS structured grid, and added rules, styles and typographic styles. More recently, we adopted Bootstrap by Twitter. This framework includes a more robust set of features such as javascript, jQuery, LESS — a dynamic stylesheet language and Gyphicons Free. All our future websites will use Bootstrap.

Bootstrap by Twitter

http://twitter.github.com/bootstrap Sleek, intuitive, and powerful, this front-end framework includes simple and flexible HTML, CSS, and Javascript for rapidly designing complex websites and developing user interface components.

Blueprint CSS

http://www.blueprintcss.org Blueprint is a CSS framework. It can help you cut down on your design/development time by giving you a solid structure on which to construct your site. The foundation of the framework is an easy-to-use layout grid; it also includes sensible typography, useful plugins, and built-in print styles.

We use Github© to store web projects and websites. Public repositories can be accessed by anyone, and users wishing to make pull requests or branch and clone repositories can sign up for a free Github account. Our Github repository can be viewed at http://github.com/UCF.

Django Web Development Framework

For complex web applications where WordPress is not suitable, we utilize the Django Web Development Framework. Written in Python, Django is a high-level web framework that is built for rapid development and clean, pragmatic design. Documentation and tutorials for getting started with Django can be found at the project website: https://www.djangoproject.com.
